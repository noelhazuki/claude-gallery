できること一覧

📁
ファイル・フォルダ管理
BOX上のファイルを言葉で操作できる
  • フォルダの中身を一覧表示する
  • ファイルを別フォルダに移動・コピー
  • フォルダを新しく作る
  • ファイルやフォルダの詳細情報を取得
  • ファイルのコメント一覧・作成
  • タスクを一覧で確認
🔍
検索・発見
BOX全体からファイルを探し出す
  • キーワードでファイルを横断検索
  • ファイル種類・拡張子で絞り込み
  • メタデータの条件でSQL的に検索
  • フォルダ名でフォルダを探す
  • 共同作業者（コラボレーター）一覧
🧠
BOX AI 分析
ファイルの中身をAIが読んで答えてくれる（ダウンロード不要）
  • 「この契約書のポイントは？」と質問
  • PDFや文書の要約を作る
  • 複数ファイルをまたいで比較・分析
  • メタデータを構造的に抽出
  • リスク評価・レビューを自動化
🏷️
メタデータ管理
ファイルに情報タグを付けて整理・検索しやすくする
  • メタデータテンプレートの一覧確認
  • テンプレートのスキーマを調べる
  • ファイル・フォルダにメタデータを書き込む
  • 新しいテンプレートを作成
  • テンプレートを更新する
👁️
プレビュー・閲覧
ファイルをダウンロードせずにチャット内で確認
  • PDF・画像・Officeファイルをその場で表示
  • 特定ページを指定して確認
  • 「この図どういう意味？」とすぐ質問
  • ファイルのテキスト内容を直接取得
⬆️
アップロード・更新
クロウくんが作ったものをそのままBOXに保存
  • 新しいファイルをBOXにアップロード
  • 既存ファイルを新バージョンで上書き
  • ファイルの名前・説明・タグを更新
  • フォルダの名前・説明を更新

実際にこんな使い方ができる

💬
「プロジェクトXのフォルダ見せて」
→ BOXの中身を一覧で表示、ファイルをその場でプレビュー
📄
「この契約書で特殊な条件ない？」
→ BOX AIがPDFを読んでリスクや要点を抽出
🔎
「先月の請求書ファイル全部探して」
→ キーワード・メタデータ検索で一発で発見
📂
「2024年のファイルを年度別フォルダに整理して」
→ フォルダ作成＋ファイル移動をまとめて自動実行
🏷️
「このPDFに顧客名と日付タグをつけて」
→ メタデータを読み取って自動でタグ付け
📊
「クロウくんが作ったまとめをBOXに保存して」
→ 分析結果や文書をそのままBOXにアップ

今現在つながってる状態なので、このチャットで「BOXの〇〇フォルダ見せて」とか「このファイル探して」って話しかけるだけで動くよ。
ダウンロード不要・BOX上で完結するから安全やし速い。
